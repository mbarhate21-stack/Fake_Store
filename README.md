# Fake Store API Testing Project

## Project Overview

This project demonstrates API Testing of the Fake Store API using Postman and Newman.

The collection covers multiple API endpoints and validates responses using automated test scripts.

## Tools Used

* Postman
* Newman
* JavaScript (Postman Test Scripts)
* GitHub

## API Endpoints Tested

### Get All Products

* Validate Status Code
* Validate Response Time
* Validate Product Data

### Get Single Product

* Validate Product Details
* Validate Product ID

### Get Categories

* Validate Category List
* Validate Response Structure

### Add Product

* Validate Product Creation
* Validate Response Data

### Update Product

* Validate Product Update
* Validate Updated Fields

## Validations Performed

* Status Code Validation
* Response Body Validation
* Response Time Validation
* Data Type Validation
